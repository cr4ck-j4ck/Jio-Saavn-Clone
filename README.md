# 🎧 **Music Streaming Platform – JioSaavn Clone**

I’ve crafted a JioSaavn-inspired music streaming platform using **pure HTML, CSS, and JavaScript**. The app fetches all its data from a local **JSON file**. Designed to deliver a seamless and fast music streaming experience, this platform is optimized for smooth playback and a responsive interface.

🌐 **Explore Live Demo**: [jio-saavn-clone.onrender.com](https://jio-saavn-clone.onrender.com/)

## ✨ **Features**

- 🎵 **Smooth, uninterrupted music playback**  
- 🎨 **Sleek and responsive UI** for all screen sizes  
- 💾 **Local storage of song data** in a lightweight JSON format  
- 📱 **Mobile-friendly layout** for on-the-go listening  

---

## 🛠 **Tech Stack**

- **HTML5** – For clean and semantic structure  
- **CSS3** – To style and make the UI responsive  
- **Vanilla JavaScript** – For interactive features  
- **Render** – Hosting and deployment  
- **JSON** – Lightweight song data storage  

---

## 🚀 **How to Run Locally**

```bash
git clone https://github.com/cr4ck-j4ck/Jio-Saavn-Clone.git
cd Jio-Saavn-Clone
# Open index.html in your browser to get started
```
